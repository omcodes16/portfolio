## 🎨 UI/UX Architecture

This portfolio isn't just a document; it's a technical demonstration of modern web capabilities:

* **The Universal 3D Canvas:** A persistent, interactive WebGL particle mesh that spans across all scroll levels, reacting to cursor parallax.
* **Cinematic Boot Sequence:** Simulates a hardware terminal boot sequence (`> Initializing Embedded Logic...`) before seamlessly transitioning into the 3D Hero section.
* **Neo-Morphic Glassmorphism:** Deep Obsidian base (`#020203`) paired with Frosted Dark Glass, Electric Indigo (`#3f51b5`) borders, and Cyber Lime (`#ccff00`) glowing accents.
* **Staggered Scroll Reveals:** Hardware-accelerated intersection observers trigger cascading animations to prevent layout thrashing and maintain 60FPS.
